This plugin makes it possible to use structured data with a simple configuration. You can also switch between structured data outputs.

== Description ==

This plugin makes it possible to use structured data!

* Article
* Breadcrumb
* Image License
* Logo
* Recipe
* Sitelinks Search box
* Video
* FAQ
* Howto
* Product

In order to use this plugin, you must purchase the right to use the plugin.
You can purchase the right here.
(https://brmk.io/xZAd)

With a relatively simple initial setup, most pages automatically output *complete* structured data.
For more information on setting up plug-ins, please check this page.
(English:https://richynokurashi.com/lp/jsonld_setting-en/, Japanese:https://richynokurashi.com/lp/jsonld_setting/)

* Article
    * You must set the thumbnail image for all pages.
    * You must set URLs of privacy page, profile page, and SNS.

* Breadcrumb
    * You can choose to show or not show the breadcrumb list.

* Logo
    * You must set "logo image".

* Recipe・Video
    * You can choose to show or not show the breadcrumb list.
    * If you can use Recipe or Video, you must set API key.


In addition, regular updates ensure that the latest plugin is always available.


== Installation ==

1. From the WP admin panel, click “Plugins” -> “Add new”.
2. In the browser input box, type “auto making JSON-LD”.
3. Select the “auto making JSON-LD” plugin and click “Install”.
4. Activate the plugin.

OR...

1. Download the plugin from this page.
2. Save the .zip file to a location on your computer.
3. Open the WP admin panel, and click “Plugins” -> “Add new”.
4. Click “upload”.. then browse to the .zip file downloaded from this page.
5. Click “Install”.. and then “Activate plugin”.
